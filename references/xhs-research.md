# 调研

拿着选题，搜集写作素材。不要凭空编造，用真实数据和案例支撑笔记。

## 何时跳过此阶段

以下情况直接进入撰写阶段：
- 用户已经提供了完整素材
- 用户说"跳过调研"或类似表达

## 调研内容

### 竞品笔记分析

搜索同选题的爆款笔记，分析它们的成功要素：

**搜索关键词：**
- 核心关键词（如「护肤」）
- 长尾关键词（如「油皮夏季护肤」）
- 场景关键词（如「通勤护肤」）

**分析维度：**
- 标题：用了什么公式？字数多少？
- 封面：什么风格？配色如何？
- 正文：结构如何？字数多少？
- 标签：用了哪些？数量多少？
- 数据：点赞/收藏/评论比例如何？

**输出格式：**

```
📋 竞品笔记分析

共分析 5 篇同类笔记

## 标题分析
- 「油皮姐妹看过来！夏季护肤不闷痘的5个秘诀」— 痛点型+数字
- 「夏天油皮怎么护肤？这3个方法救了我」— 悬念型+数字
- 「油皮夏季护肤攻略｜从早到晚不脱妆」— 干货型

## 封面分析
- 对比图：3 篇（60%）
- 大字报：2 篇（40%）
- 主色调：白色、粉色

## 正文分析
- 平均字数：480 字
- 结构：场景代入→分步骤→总结
- emoji：每段 2-3 个

## 标签分析
- 高频标签：#护肤 #油皮 #夏季
- 标签数量：5-8 个

## 💡 可借鉴元素
1. 标题用「姐妹」拉近距离
2. 封面用对比图最直观
3. 正文分步骤，每步一个 emoji
```

### 素材搜集

根据大纲，按每个章节搜集写作素材。

**素材来源：**

| 来源 | 搜什么 | 怎么搜 |
|------|--------|--------|
| 小红书笔记 | 同类爆款、用户评价 | anysearch 搜索关键词 |
| 产品官网 | 产品参数、官方说明 | WebFetch 提取页面 |
| 用户评价 | 真实体验、优缺点 | 搜索「XX 评价」「XX 测评」 |
| 行业报告 | 数据、趋势 | 搜索「XX 行业报告」「XX 数据」 |
| KOL 内容 | 专业观点、使用技巧 | 搜索「XX 博主」「XX 达人」 |

**素材类型：**
- 数据：具体数字、统计结果
- 案例：真实故事、使用体验
- 引用：专家观点、用户评价
- 对比：产品对比、前后对比
- 价格：购买渠道、价格信息

### 结构化素材包

搜集完成后，整理成结构化素材包：

```
📝 素材包：油皮夏季护肤指南

## 数据素材
- 根据 XX 报告，油皮占比 35%
- 小红书「油皮护肤」话题浏览量 2.3 亿

## 案例素材
- 用户 A：「用了 XX 方法，一周后出油明显减少」
- 用户 B：「之前用 XX 产品，反而更油了」

## 产品素材
- XX 面霜：价格 189 元，适合油皮，质地清爽
- XX 精华：价格 259 元，控油效果好

## 对比素材
- 使用前：油光满面、毛孔粗大
- 使用后：清爽干净、毛孔细腻

## 价格素材
- 平价选择：50-100 元
- 中端选择：100-300 元
- 高端选择：300 元以上

## 来源标注
- 数据来源：XX 报告、小红书官方
- 案例来源：小红书用户评价
- 产品来源：品牌官网
```

## 调研技巧

### 搜索技巧

1. **关键词组合**：用「品类+场景」「人群+痛点」组合搜索
2. **时间限定**：优先选择近 30 天内的内容
3. **热度排序**：按互动量排序，找爆款
4. **多平台搜索**：小红书 + 微博 + 知乎 + 豆瓣

### 素材筛选

1. **真实性**：优先选择有真实体验的内容
2. **时效性**：优先选择近期内容
3. **数据支撑**：优先选择有具体数据的内容
4. **可验证**：优先选择可交叉验证的内容

### 素材整理

1. **分类存储**：按类型（数据/案例/产品/对比）分类
2. **标注来源**：每个素材标注来源，方便引用
3. **去重合并**：相同素材合并，避免重复
4. **标记可用性**：标记哪些素材可以直接用，哪些需要改写

## 半自动 vs 全自动

**半自动：**
- 展示竞品分析结果，问用户是否补充
- 展示素材清单，问用户是否需要更多
- 问用户是否有自己的素材要加入

**全自动：**
- 自动完成竞品分析
- 自动搜集素材并整理
- 直接进入写稿阶段

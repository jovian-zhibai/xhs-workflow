# 增强工具集

随时可调用的工具，不绑定特定流程阶段。用户说"帮我想几个标题"或"帮我检查一下笔记"时使用。

## 爆款标题生成器

基于笔记内容生成多个标题选项，帮用户找到最有吸引力的那个。

### 何时使用

- 用户写完笔记后不知道怎么取标题
- 用户对现有标题不满意
- 用户说"帮我想几个标题"或"标题太普通了"

### 怎么做

1. 通读笔记全文，理解核心观点和目标读者
2. 用以下 6 种策略各生成 2 个标题，共 12 个：
   - **数字法**：用具体数字降低门槛（「5 个方法」「3 天见效」）
   - **痛点法**：直击读者焦虑（「你还在 XX？」「为什么你总是 Y」）
   - **悬念法**：制造信息缺口（「这个方法让我月入 X」「90% 的人不知道」）
   - **对比法**：制造反差（「从 X 到 Y，我只做了一件事」）
   - **情绪法**：引发共鸣（「后悔没早知道！这些太香了」）
   - **干货法**：承诺价值（「完整教程」「保姆级攻略」「必看指南」）
3. 每个标题必须与笔记核心内容直接相关，不能为了爆款而偏离原文
4. **每个标题 ≤ 20 字（含 emoji 和标点）**：生成后必须数一遍，超过的立即截断或重写。这是小红书硬限制，超限无法发布
5. 避免 AI 味标题：不用「震惊」「必看」「深度解析」等套路词
6. 每个标题附带一句话分析：为什么这样写、适合什么读者
7. 按推荐度排序，最推荐的放前面

**被否定后：** 如果用户否定一轮标题，先问"你想要什么感觉的标题？"（比如更口语、更正式、更夸张、更低调），再针对性生成

### 输出格式

```
🏷️ 爆款标题候选（共 12 个）

⭐ 推荐
1. 「油皮姐妹看过来！夏季护肤不闷痘的5个秘诀」— 痛点法+数字，精准定位油皮人群
2. 「夏天油皮怎么护肤？这3个方法救了我」— 悬念法，制造好奇心

📌 备选
3. 「从油光满面到清爽干净，我只做了这一件事」— 对比法，前后反差吸引点击
4. 「油皮夏季护肤攻略｜从早到晚不脱妆」— 干货法，承诺完整解决方案
...

💡 分析：这篇笔记的核心价值是 XX，建议标题突出 XX，避免 XX。
```

## 标签推荐器

为笔记推荐合适的标签组合，采用金字塔搭配策略。

### 何时使用

- 用户写完笔记需要标签
- 用户说"帮我推荐标签"
- 在发布前自动生成

### 怎么做

1. 通读笔记，提取核心关键词
2. 按金字塔结构推荐标签：
   - **热门大标签**（1-2 个）：百万级浏览量，获取曝光
   - **精准中标签**（2-3 个）：十万级浏览量，精准触达
   - **长尾小标签**（2-3 个）：万级浏览量，降低竞争
3. 标签必须与内容强相关
4. 推荐 5-10 个标签

**标签类型：**
- 品类标签：#护肤 #穿搭 #美食
- 场景标签：#通勤穿搭 #学生党平价
- 情绪标签：#好物分享 #生活记录
- 时效标签：#618好物 #双11攻略

### 输出格式

```
🏷️ 标签推荐（共 8 个）

🔥 热门标签（获取曝光）
1. #护肤 — 浏览量 120 亿
2. #好物分享 — 浏览量 85 亿

🎯 精准标签（精准触达）
3. #油皮护肤 — 浏览量 5.2 亿
4. #夏季护肤 — 浏览量 3.8 亿
5. #学生党平价 — 浏览量 2.1 亿

📍 长尾标签（降低竞争）
6. #油皮不闷痘 — 浏览量 8000 万
7. #夏季控油 — 浏览量 5000 万
8. #油皮护肤步骤 — 浏览量 3000 万

💡 建议：标签数量 5-10 个最佳，太少覆盖搜索词不足，太多稀释核心标签权重。
```

## 笔记体检报告

分析笔记的可读性和种草力，给出具体优化建议。

### 何时使用

- 用户写完笔记想优化
- 用户说"帮我看看这篇笔记怎么样"
- 用户说"有没有什么可以改进的"
- 在同步发布之前做一次体检

### 怎么做

从以下 5 个维度打分（每项 1-5 分），并给出具体改进建议：

| 维度 | 检查什么 |
|------|---------|
| **封面吸引力** | 封面是否清晰、美观、信息明确？是否能在 3 秒内传递核心信息？ |
| **标题点击率** | 标题是否有吸引力？是否包含关键词？是否用了爆款公式？ |
| **正文种草力** | 正文是否有真实感？是否有细节？是否有对比？是否能引发共鸣？ |
| **标签精准度** | 标签是否与内容相关？是否覆盖核心关键词？金字塔搭配是否合理？ |
| **互动引导力** | 结尾是否有互动引导？是否能引发评论和收藏？ |

### 输出格式

```
📊 笔记体检报告

总分：42/50 ⭐⭐⭐⭐

## 封面吸引力：8/10
✅ 封面清晰、美观
⚠️ 建议增加文字信息，让用户一眼看到核心卖点

## 标题点击率：9/10
✅ 标题包含关键词「油皮」「夏季」
✅ 使用了数字「5个」降低门槛
⚠️ 可以增加 emoji 增加视觉吸引力

## 正文种草力：8/10
✅ 有真实使用体验
✅ 有具体产品推荐
⚠️ 可以增加前后对比图
⚠️ 可以增加价格信息

## 标签精准度：9/10
✅ 标签与内容强相关
✅ 金字塔搭配合理
⚠️ 可以增加一个时效标签（如 #618好物）

## 互动引导力：8/10
✅ 结尾有提问引导
⚠️ 可以增加收藏引导语

## 💡 优化建议
1. 封面增加文字信息
2. 标题增加 emoji
3. 正文增加前后对比图
4. 正文增加价格信息
5. 增加收藏引导语
```

## 开头 3 行生成器

小红书折叠后只显示前 3 行，决定是否展开。

### 何时使用

- 用户不知道怎么写开头
- 用户说"帮我优化一下开头"
- 体检报告指出开头吸引力不足

### 怎么做

1. 通读笔记，理解核心卖点
2. 生成 3 个版本的开头：
   - **场景代入版**：描述用户熟悉的场景
   - **痛点提问版**：直击用户痛点
   - **数据冲击版**：用具体数字吸引注意
3. 每个版本控制在 3 行以内
4. 确保能引发继续阅读的欲望

### 输出格式

```
📝 开头 3 行候选（共 3 个）

1️⃣ 场景代入版
夏天一出门就油光满面？
洗完脸不到 2 小时就出油？
油皮姐妹的痛我太懂了！

2️⃣ 痛点提问版
油皮夏季护肤最大的误区是什么？
就是过度控油！
今天教你 5 个方法，清爽一整天～

3️⃣ 数据冲击版
90% 的油皮都在用错护肤品！
我之前也是，直到发现了这 5 个方法...
现在夏天出门一整天都不油！

💡 建议：如果你的笔记偏教程类，选痛点提问版；如果偏种草类，选场景代入版。
```

## 封面设计指导

根据笔记类型推荐封面风格。

### 何时使用

- 用户不知道怎么做封面
- 用户说"帮我设计封面"
- 在排版阶段自动推荐

### 封面风格库

| 风格 | 特点 | 适合笔记类型 |
|------|------|-------------|
| **对比图** | 前后效果对比，转化率最高 | 种草、测评、教程 |
| **大字报** | 核心卖点大字展示 | 干货、攻略、合集 |
| **拼贴** | 多图拼接、活泼 | 合集、清单、穿搭 |
| **渐变** | 柔和、年轻感 | 生活方式、情感 |
| **清单** | 编号列表、信息密度高 | 清单、攻略、教程 |

### 设计要点

**尺寸：** 3:4 竖版（1080×1440px）

**构图：**
- 主标题：占画面上方 1/3，字号最大
- 副标题/分类标签：居中，字号中等
- 案例图/示意图：底部，作为视觉补充

**配色：**
- 对比鲜明，不超过 3 种颜色
- 背景色与文字色对比强烈
- 核心信息一眼可见

**禁忌：**
- 模糊不清的图片
- 与内容不符的封面
- 过度 PS/滤镜导致失真
- 满屏文字无视觉焦点

### 输出格式

```
🎨 封面设计方案

推荐风格：对比图

构图建议：
- 左图：使用前（油光满面）
- 右图：使用后（清爽干净）
- 文字：大字标题叠加

配色建议：
- 主色：白色
- 强调色：粉色
- 文字色：黑色

模板推荐：templates/bold.html（大字报风）

💡 提示：可以截图产品官网或用户评价作为素材。
```

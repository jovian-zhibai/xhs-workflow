# 封面图模板

5 个小红书封面图 HTML 模板，所有样式均为内联样式，可直接在浏览器中截图使用。

## 设计原则

- **3:4 竖版** — 1080×1440px，瀑布流中占屏比最大
- **内联样式** — 所有样式使用 inline style，不依赖外部 CSS
- **无背景图** — 使用纯色背景，避免图片加载问题
- **文字为主** — 封面核心是文字信息，图片作为辅助
- **对比鲜明** — 文字与背景对比强烈，确保可读性

## 模板列表

| 模板 | 文件 | 风格 | 适合笔记类型 |
|------|------|------|-------------|
| 简约白 | `minimal.html` | 干净、留白、高级感 | 干货、教程、清单 |
| 大字报 | `bold.html` | 大字、冲击力、吸睛 | 种草、测评、攻略 |
| 拼贴 | `collage.html` | 多图拼接、活泼 | 合集、穿搭、美食 |
| 渐变 | `gradient.html` | 柔和、年轻、时尚 | 生活方式、情感、故事 |
| 清单 | `list.html` | 编号列表、信息密度高 | 清单、攻略、教程 |

## 模板选择策略

**全自动模式：** AI 根据爆款拆解结果自动选择最匹配的模板

| 爆款拆解结果 | 推荐模板 |
|-------------|---------|
| 对比图占比最高 | 大字报、简约白 |
| 大字报占比最高 | 大字报 |
| 拼贴占比最高 | 拼贴 |
| 信息密度高 | 清单、简约白 |
| 情感类内容 | 渐变 |

**半自动模式：** 列出模板选项让用户选择

## 使用方式

每个模板有以下占位符：

- `{{TITLE}}` — 主标题（必填）
- `{{SUBTITLE}}` — 副标题（可选）
- `{{CATEGORY}}` — 分类标签（可选）
- `{{ITEM_1}}` ~ `{{ITEM_5}}` — 列表项（清单模板专用）

使用方式：
1. 根据笔记内容替换占位符
2. 在浏览器中打开 HTML 文件
3. 截图保存为图片（建议使用浏览器开发者工具调整为 1080×1440px）

## 设计要点

### 文字层级

- 主标题：占画面上方 1/3，字号最大（≥36pt）
- 副标题/分类标签：居中，字号中等
- 辅助文字：底部，字号较小

### 配色建议

| 笔记类型 | 主色 | 强调色 | 文字色 |
|---------|------|--------|--------|
| 护肤美妆 | 粉色 #FFB6C1 | 玫红 #FF69B4 | 黑色 #333 |
| 穿搭时尚 | 黑色 #333 | 金色 #FFD700 | 白色 #FFF |
| 美食 | 橙色 #FFA500 | 红色 #FF4500 | 白色 #FFF |
| 旅行 | 蓝色 #4A90D9 | 青色 #00CED1 | 白色 #FFF |
| 学习成长 | 绿色 #32CD32 | 深绿 #228B22 | 白色 #FFF |

### 截图技巧

1. 使用 Chrome 浏览器打开 HTML 文件
2. 按 F12 打开开发者工具
3. 点击左上角的「设备模拟」图标
4. 设置分辨率为 1080×1440
5. 截图保存

## 注意事项

- 封面图不要使用真人照片（平台限制）
- 文字不要过多，3 行以内最佳
- 确保文字与背景对比强烈
- 核心信息一眼可见
- 避免模糊不清的设计
